**Please refer, **

This example is full function and usable for Arduino IDE.
(Test on Arduino UNO, Dev, major esp32 MCUs with WIFI function)

Assign and review your i2c Pins, which are connected correctly between sensor and MCU.

Installation:
For Arduino IDE, extract it in your Arduino or other IDE libraries Path:
Library path for Arduino is: C:\Users\user_name\Documents\Arduino\libraries\KW007B
Open the KW007_Dashboard.ino files, it will prompt it need to be in a folder, and it will do it for you.

(*Arduino IDE may ask you for "WifiManager by Tzapu" which is from the Library Manager in the IDE, current version are 2.0.17, provide easy wifi connection,  all are free and safe to use commercially. )

Read the code and change the i2c pins if you needed, there are few options of connection too, upload to your board when you found your i2c pin is match with your MCU.

Default connection :
Power on the MCUs, using phone or laptop search and connect for 
WIFI SSID : KW007-Radar; Password is 12345678

Using browser input URL 192.168.4.1 or KW007.local

You will see the control panel on your screen, it's data stream into your phone and most of the settings are instant effective/apply.

Let us know if you have any issues, maybe a different MCU, or want it on BLE, we can't help with that.
